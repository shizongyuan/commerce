# 技能文件：售后服务

## 技能基础信息
- **技能名称**: 售后服务
- **技能编号**: SKILL-AFTER-SALES-001
- **版本**: v1.0
- **状态**: 启用

## 触发词
- "售后"、"退货"
- "换货"、"维修"
- "坏了"、"质量问题"

## 执行流程

### 步骤1: 问题确认
了解产品问题和购买时间。

### 步骤2: 方案提供
| 问题类型 | 处理方案 |
|----------|----------|
| 7天内退货 | 全额退款 |
| 7-15天换货 | 换同款或等价商品 |
| 15天以上 | 维修或补贴 |
| 质量问题 | 无条件退换 |

### 步骤3: 操作指引
指导客户提交售后申请。

### 步骤4: 进度跟踪
记录工单，安排取件或维修。

## 常见问答

| 问题 | 回复 |
|------|------|
| 可以退货吗？ | 您好，自收货之日起7天内可申请退货 |
| 怎么申请售后？ | 您可以在订单详情页提交售后申请 |
| 退货运费谁承担？ | 质量问题由我们承担，非质量问题由您承担 |
