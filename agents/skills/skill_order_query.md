# 技能文件：订单查询

## 技能基础信息
- **技能名称**: 订单查询
- **技能编号**: SKILL-ORDER-001
- **版本**: v1.0
- **状态**: 启用
- **最后更新**: 2026-05-01

## 触发条件

### 触发词
- "查订单"、"我的订单"、"订单在哪"
- "发货了吗"、"什么时候发货"
- "订单状态"

### 排除词
- "取消订单" → 触发退款技能
- "修改地址" → 转人工

## 执行流程

### 步骤1: 身份验证
要求客户提供手机号后4位或订单号进行验证。

### 步骤2: 订单查询
调用 query_order 工具查询订单。

### 步骤3: 状态解读
| 状态 | 回复策略 |
|------|---------|
| 待付款 | 提供支付链接，引导付款 |
| 已付款待发货 | 告知预计发货时间 |
| 已发货 | 查询物流进度 |
| 已签收 | 引导确认收货+好评 |
| 退货退款中 | 告知退款流程 |
