# 技能文件：物流查询

## 技能基础信息
- **技能名称**: 物流查询
- **技能编号**: SKILL-LOGISTICS-001
- **版本**: v1.0
- **状态**: 启用

## 触发词
- "物流"、"快递"
- "发货了吗"、"到哪了"
- "查快递"、"物流信息"
- "什么时候到"

## 执行流程

### 步骤1: 身份验证
要求提供订单号或手机号后4位。

### 步骤2: 物流查询
根据订单号查询物流信息。

### 步骤3: 状态解读
| 状态 | 说明 |
|------|------|
| 已发货 | 告知快递公司和运单号 |
| 运输中 | 告知当前所在地 |
| 派送中 | 预计送达时间 |
| 已签收 | 确认收货人 |

### 步骤4: 异常处理
- 物流延迟 → 联系快递公司核实
- 丢件 → 启动理赔流程

## 常见问答

| 问题 | 回复 |
|------|------|
| 发货了吗？ | 您好，请提供订单号我帮您查询 |
| 到哪了？ | 根据物流信息，您的包裹目前在[地点] |
| 什么时候到？ | 预计[时间]送达 |
