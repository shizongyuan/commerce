# 技能文件：投诉处理

## 技能基础信息
- **技能名称**: 投诉处理
- **技能编号**: SKILL-COMPLAINT-001
- **版本**: v1.0
- **状态**: 启用

## 触发词
- "投诉"、"差评"
- "不满"、"太差了"
- "要举报"

## 执行流程

### 步骤1: 情绪安抚
表达歉意和理解，缓解客户情绪。

### 步骤2: 问题收集
了解订单号、问题描述、期望解决方案。

### 步骤3: 问题分类
- 物流问题 → 查物流+补偿
- 产品质量 → 退换货
- 服务态度 → 道歉+反馈

### 步骤4: 升级处理
无法解决时转人工，记录工单。

## 补偿标准
- 物流延迟: 订单金额5%，最高50元
- 产品质量: 订单金额10-30%
- 服务态度: 10-20元优惠券
