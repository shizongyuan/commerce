# 技能文件：产品咨询

## 技能基础信息
- **技能名称**: 产品咨询
- **技能编号**: SKILL-PRODUCT-001
- **版本**: v1.0
- **状态**: 启用

## 触发词
- "这个产品"、"产品介绍"
- "好不好用"、"效果怎么样"
- "适合我吗"

## 执行流程

### 步骤1: 了解需求
询问客户肤质/需求，以便推荐合适产品。

### 步骤2: 产品推荐
根据知识库产品知识库推荐合适产品。

### 步骤3: 后续引导
- 主动告知价格和优惠
- 引导到产品页或留资
