# 护肤产品知识库

## 产品分类

### 洁面系列
- **洁面乳**：温和清洁，适合日常使用
- **洁面霜**：深层清洁，适合浓妆后使用
- **洁面慕斯**：绵密泡沫，适合敏感肌

### 精华系列
- **保湿精华**：玻尿酸成分，深层补水
- **美白精华**：VC衍生物，淡化色斑
- **抗衰精华**：视黄醇，促进胶原蛋白

### 面霜系列
- **日霜**：含防晒指数，日间防护
- **晚霜**：修护肌肤，夜间滋养
- **防晒霜**：SPF50+，强效防晒

## 适用肤质
| 肤质 | 推荐产品 |
|------|---------|
| 干性肌肤 | 保湿精华 + 晚霜 |
| 油性肌肤 | 洁面慕斯 + 控油精华 |
| 敏感性肌肤 | 温和洁面 + 舒缓面霜 |
| 混合性肌肤 | 分区护理，T区控油 |

## 使用顺序
1. 洁面 → 2. 爽肤水 → 3. 精华 → 4. 面霜 → 5. 防晒（白天）

## 常见问题
- Q: 护肤品需要多久见效？
  A: 一般坚持使用 4-8 周可见明显效果

- Q: 可以同时使用多种精华吗？
  A: 建议最多叠加两种不同功效的精华

- Q: 孕期可以使用吗？
  A: 请查看成分表，含视黄醇、 水杨酸等产品建议慎用
